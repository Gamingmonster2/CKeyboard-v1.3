# CKeyboard v1.3 ⌨️🎙️

**CKeyboard** is a lightweight, smart web-based keyboard tailored for seamless typing and voice-to-text input, with full support for the Arabic language. Built with minimalism and performance in mind.

---

## 🌟 المميزات (Features)
- **دعم اللغة العربية:** لوحة مفاتيح متكاملة للكتابة باللغة العربية.
- **الإدخال الصوتي (Voice Typing):** تحويل الكلام إلى نص بدقة عالية باستخدام واجهات الويب (Web Speech API).
- **تصميم خفيف (Lightweight):** مبني بالكامل في ملف `index.html` واحد لضمان أقصى سرعة تحميل بدون مكتبات خارجية ثقيلة.
- **واجهة مستخدم عصرية (UI/UX):** تصميم بسيط (Minimalist) مريح للعين يناسب بيئات العمل الاحترافية.

## 🛠️ التثبيت المحلي (Local Usage)
نظراً لبساطة المشروع، لا تحتاج إلى أي إعدادات معقدة أو خوادم (Servers):
1. قم بتحميل المستودع (Clone the repository).
2. افتح ملف `index.html` مباشرة في أي متصفح حديث (يفضل Chrome أو Edge لضمان عمل ميزة الإدخال الصوتي بكفاءة).

## 🛡️ الخصوصية والأمان (Privacy & Security)
تعمل هذه الأداة بالكامل من جهة العميل (Client-Side). لا يتم إرسال أو حفظ أي نصوص أو تسجيلات صوتية على أي خوادم خارجية. خصوصيتك محفوظة 100%.
